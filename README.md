# Matomo Password Policy Enforcer plugin
Password Policy plugin for Matomo (aka Piwik).

## Installation
Install it via Matomo Marketplace

| Matomo version | Plugin version |
|----------------| ------------- |
| 3.x            | 1.x |
| 4.x            | 2.x |
| 5.x            | 2.x |
